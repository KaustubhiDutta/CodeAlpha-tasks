# 🚀 AI Compass – AI Learning & Discovery Platform

AI Compass is an intelligent AI-powered learning and discovery platform that helps users explore Artificial Intelligence concepts, discover AI tools, compare platforms, and learn through an interactive NLP chatbot powered by Cosine Similarity.

🌐 **Live Demo:** https://code-alpha-tasks-chat.vercel.app

---

## 📌 Features

### 🤖 Intelligent NLP Chatbot
- Natural Language Processing with Cosine Similarity
- Fast semantic search for AI-related queries
- Interactive conversational interface
- Confidence-based responses
- Related question suggestions

### 📚 AI Learning Hub
- Beginner-friendly AI topics
